FREQUENTLY ASKED QUESTIONS

How is file security calculated locally in my browser?
We load file headers into local ArrayBuffer indexes using standard HTML5 FileReader APIs, calculate precise cryptographic SHA-256 digests via browser WebCrypto APIs, and verify integrity byte blocks without forwarding any data bytes to external cloud systems.
What is Steganography and LSB encoding?
Least Significant Bit (LSB) steganography embeds binary message bytes into the lower bit indexes of image pixel colors (R, G, B channels). This introduces microscopic color alterations that are completely imperceptible to the naked human eye but instantly recoverable by decryption algorithms.
How safe is the URL scanner?
The URL inspector runs heuristic pattern-matching to flag suspicious subdomains, double extensions, homograph ASCII layouts, missing secure protocol wrappers (HTTP vs HTTPS), and high-frequency phishing brand keywords before you pivot your navigation to them.
